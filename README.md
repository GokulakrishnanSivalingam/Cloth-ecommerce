## üì¶ Installation

To get the project running on your local machine:

```bash
git clone https://github.com/GokulakrishnanSivalingam/cloth-ecom-site.git
cd cloth-ecom-site
npm install

# Start client
cd client
npm install
npm run dev

# Start server
cd ../server
npm init -y
npm i express mongoose razorpay 
npm run dev
```
